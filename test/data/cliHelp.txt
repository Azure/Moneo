  -d {manager,workers,full}, --deploy {manager,workers,full}
                        Deployment choices: {manager,workers,full}. Requires config file to be specified (i.e. -c host.ini) or file to be in Moneo directory.
  -s {manager,workers,full}, --shutdown {manager,workers,full}
                        Shutdown choices: {manager,workers,full}. Requires config file to be specified (i.e. -c host.ini) or file to be in Moneo directory.
  -c HOST_INI, --host_ini HOST_INI
                        Provide filepath and name of ansible config file. The default is host.ini in the Moneo directory.
